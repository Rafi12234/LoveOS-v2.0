# LoveOS v2.0

A developer-themed anniversary web app built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Boot screen with typewriter animation
- Interactive terminal with custom commands
- Relationship commit timeline
- Memory gallery with categories
- Love letter with decryption effect
- Future roadmap with progress checklist
- Live countdown / elapsed time counter
- Confetti celebrations
- Fully responsive design

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- Canvas Confetti

## Getting Started

```bash
npm install
npm run dev
```
